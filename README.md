## Overview
The script randomly selects letters from the English alphabet (uppercase and lowercase) until the letter "w" is drawn.
It prints every randomly chosen letter and counts how many draws were needed.

## Usage
```bash
python tp01.py
```

## Requirements
- Python 3.x

## Features
-Random letter generation

-Case-sensitive comparison

-Infinite loop protection via condition (c != "w")

-Counts and prints all attempts
